
ELEMENT

<!ELEMENT elemento EMPTY>                      "El elemento no puede tener contenido."

<!ELEMENT elemento ANY>                        "Puede contener cualquier tipo de contenido."

<!ELEMENT elemento (#PCDATA)>                  "Contenido solo de texto."

<!ELEMENT elemento (hijo)>                     "El elemento contiene exactamente al elemento hijo (una vez)."

<!ELEMENT elemento (hijo+)>                    "El '+' significa uno o más elementos 'hijo'."

<!ELEMENT elemento (hijo*)>                    "El '*' es cero o más veces."

<!ELEMENT elemento (hijo?)>                    "El '?' es opcional, cero o una vez."

<!ELEMENT elemento (a, b, c)>                  "Los elementos deben aparecer en ese orden exacto."

<!ELEMENT elemento (a | b | c)>                "Solo puede aparecer uno de ellos (alternativa)."

<!ELEMENT elemento (a, (b | c)?, d*)>          "Se pueden combinar los operadores.", "a" es obligatorio, b/c son opcionales, perimero b o primero c pero a la vez no, y d: 0 o mas

<!ELEMENT dades ((cognom,nom)|(nom,cognom))>   "El elemento admite dos secuencias posibles: primero cognom y luego nom, o primero nom y luego cognom."





ATTLIST

<!ATTLIST elem attr CDATA #REQUIRED>             "Atributo obligatorio de texto."

<!ATTLIST elem attr CDATA #IMPLIED>              "Atributo opcional."

<!ATTLIST elem attr CDATA "valor">               "Atributo con valor por defecto."

<!ATTLIST elem attr ID #REQUIRED>                "ID único en todo el documento."

<!ATTLIST elem attr IDREF #REQUIRED>             "Referencia a otro ID existente."

<!ATTLIST elem attr (a | b | c) #REQUIRED>       "Enumeración: solo puede ser uno de esos valores."

<!ATTLIST elem attr CDATA #FIXED "valor">        "El atributo siempre tendrá ese valor y no puede cambiar."

<!ATTLIST elem href CDATA 'http://'>             "Atributo de texto con valor por defecto 'http://'. Si no se especifica en el XML, usa ese valor."

<!ATTLIST elem attr IDREF #IMPLIED>              "Atributo opcional que debe contener una referencia a un ID existente en el documento."

<!ATTLIST persona contr (true|false) 'false'>    "Atributo con enumeración; solo puede ser true o false. Si no se indica, toma el valor por defecto 'false'."
